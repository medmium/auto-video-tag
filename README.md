## 使用方法: 

切换到你的插件目录
```
cd /path/to/your/vault/.obsidian/plugins
```
拉取本项目
```
git clone https://github.com/medmium/auto-video-tag.git
```

回到obsidian启用第三方插件, 即可启用mp4附件md链接转换为html标签

## 更好的选择

如果你是撰写自己的博客网站,苦恼于网站不能显示附件视频, 请尽量不要使用本插件, 我已经为基于jekyll生成了一个自动处理脚本,详情请[查看博客].(http://blog.medmi.cn/posts/%E5%88%9B%E5%BB%BAjekyll%E6%8F%92%E4%BB%B6%E4%BB%A5%E8%87%AA%E5%8A%A8%E8%BD%AC%E6%8D%A2MP4/)
